# ブックアプリ

読んだ本を記録し、評価と感想を残すシンプルなWebアプリです。

## 主な機能
- タイトルと感想を登録
- 星評価
- お気に入り（いいね）
- 読了管理
- localStorageによる保存

## APIについて
書籍画像・価格取得には外部APIを使用していますが、  
APIキー保護のため **公開版では無効化** しています。

## 使用技術
- HTML
- CSS
- JavaScript

## 開発について
本アプリは、設計と考慮を重視しながら、  
一部の実装をAIと対話して作成しました。
